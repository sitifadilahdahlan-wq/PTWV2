Informasi Perhitungan

Metodologi dan definisi untuk Safety Key Metrics

Frequency Rate (FR)

FR = (Jumlah kecelakaan dengan Lost Time × 1.000.000) / Total jam kerja

Severity Rate (SR)

SR = (Jumlah hari hilang × 1.000.000) / Total jam kerja

Fatal Accident Rate (FAR)

FAR = (Jumlah fatality × 100.000.000) / Total jam kerja

Hubungkan Google Spreadsheet

Integrasikan form web app portal AK4L secara real-time ke lembar kerja Anda.

Masukkan URL Web App Apps Script Anda. Agar tampilan aplikasi terbuka penuh, pastikan fungsi doGet(e) mengembalikan HtmlService.

Script Google Apps Script (Code.gs)

Gunakan kode ini pada editor Code.gs agar aplikasi dapat dibuka langsung dari URL Apps Script tanpa menampilkan teks polos.

Petunjuk Deploy Tampilan Web App:
  1. Buat file HTML baru di Google Apps Script bernama index dan tempelkan seluruh kode HTML aplikasi ini.
  2. Salin kode Code.gs di bawah ini ke file Code.gs.
  3. Klik Deploy > New Deployment > Pilih type Web app > Execute as: Me > Who has access: Anyone.
function doGet(e) {
  // Render tampilan web app HTML langsung saat URL dibuka di browser
  return HtmlService.createHtmlOutputFromFile('index')
      .setTitle('AK4L - QSHE & Security Portal | LRT Jakarta')
      .setXFrameOptionsMode(HtmlService.XFrameOptionsMode.ALLOWALL)
      .addMetaTag('viewport', 'width=device-width, initial-scale=1');
}

function doPost(e) {
  try {
    var data = JSON.parse(e.postData.contents);
    var result = processAction(data.action, data);
    return ContentService.createTextOutput(JSON.stringify(result))
      .setMimeType(ContentService.MimeType.JSON);
  } catch (err) {
    return ContentService.createTextOutput(JSON.stringify({ status: 'error', message: err.message }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

function processAction(action, data) {
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  var sheetName = "Data_" + action;
  var sheet = ss.getSheetByName(sheetName) || ss.insertSheet(sheetName);
  
  if (sheet.getLastRow() === 0) {
    sheet.appendRow(Object.keys(data));
  }
  
  var row = [];
  for (var key in data) {
    row.push(data[key]);
  }
  sheet.appendRow(row);
  
  return { status: "success", action: action };
}
